Легко изменяйте курсор на сайте. В плагине SA Custom Cursor есть более 20 гибких настроек. Можно настроить как основной курсор, так и второстепенный.

== Description ==

Плагин для изменения курсора на сайте.

С этим плагином вы сможете с легкостью изменить курсор на вашем сайте.

== Frequently Asked Questions ==

= Плагин полностью бесплатный? =

Да, плагин не требует никаких дополнительных плат.

= Какие преимущества данного плагина? =

Плагин имеет множество настроек, вплоть, до добавление изображения за курсором.
